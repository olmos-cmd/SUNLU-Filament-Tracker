# SUNLU Filament Tracker 1.10.1

## English

First public GitHub release of the SUNLU Filament Tracker for Windows.

### Highlights

- Portable Windows application written in Rust
- German and English interface
- Dark user interface with custom program icon
- Filament spool management with remaining-weight display
- AMS unit and slot assignment
- Import of normal and sliced 3MF files
- Estimated consumption for normal 3MF files
- Stored slicer values used when available
- Print history and CSV export
- SQLite database with backup and restore
- Automatic retention of up to ten backups

This software is freeware. Please read the included `LICENSE` before redistribution or modification.

## Deutsch

Erste öffentliche GitHub-Veröffentlichung des SUNLU Filament Trackers für Windows.

### Höhepunkte

- portable Windows-Anwendung in Rust
- deutsche und englische Benutzeroberfläche
- dunkle Oberfläche mit eigenem Programmsymbol
- Verwaltung von Filamentrollen mit Restbestandsanzeige
- AMS-Einheit und Fachzuordnung
- Einlesen normaler und geslicter 3MF-Dateien
- Verbrauchsschätzung bei normalen 3MF-Dateien
- Übernahme gespeicherter Slicerwerte, sofern vorhanden
- Druckhistorie und CSV-Export
- SQLite-Datenbank mit Sicherung und Wiederherstellung
- automatische Aufbewahrung von maximal zehn Backups

Dieses Programm ist Freeware. Vor Weitergabe oder Änderung bitte die enthaltene `LICENSE` lesen.
