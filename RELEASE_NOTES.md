# Release notes / Versionshinweise

## SUNLU Filament Tracker 1.10.1

### English

Version 1.10.1 focuses on the GitHub publication, documentation, release preparation, and general reliability improvements.

- Revised GitHub project page and documentation
- Added and updated README, license information, and release notes
- Improved remaining-filament calculation
- Optimized 3MF consumption analysis
- Updated print history, statistics, CSV export, backup, and restore documentation
- Added further fixes and internal optimizations
