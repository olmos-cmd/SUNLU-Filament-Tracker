# Changelog / Änderungsverlauf

## English

### Version 1.10.1
- Permanent dark design
- Light mode removed
- Fixed footer with version, database name, and status
- Window height adjusted for clean display above the Windows taskbar
- Small header logo optimized for improved readability
- German and English interface
- Expanded About page with program description, freeware information, rights, and liability notice
- Added explanation of the filament consumption calculation

### Version 1.9.x
- Price comparison feature completely removed
- Improved database backup and restore
- Limited backups in the roaming directory to a maximum of ten
- Revised spool overview and print-file section
- Adjusted CSV export for German special characters

### Version 1.8.x
- Added new spool graphics and dark dashboard
- Expanded material and color selection
- Improved 3MF geometry analysis and filament consumption estimation
